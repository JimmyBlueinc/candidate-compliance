## ğŸ“¡ How Requests Work

### Frontend Requests

1. Frontend calls: `http://cpdemo.blueinctech.com/api/login`
2. Web server routes to: `api/index.php`
3. Laravel processes request using files from: `backend/` folder
4. Returns JSON response to frontend
